Для работы модуля необходимо установить пакет через NuGet:
System.Text.Json


CurrencyService 
Работает с API Центробанка. Поддерживает "USD", "KZT", "EUR" и др.


NotificationService
Отправляет чек на почту клиента через SMTP Mail.ru


TelegramService
Отправляет отчет о заказе в Telegram-бот. 
   Сейчас настроен на мой ID. Сообщения будут приходить мне на телефон.
